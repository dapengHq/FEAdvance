# HTML #
## HTML5的新特性
- 新标签
    - HTML5中的区块和段落元素
        - header    [展示介绍性内容，通常包含标题、搜索栏、logo、作者名称等]
        - nav   [在文档中提供导航链接]
        - section   [表示页面中可复用的独立结构，如帖子、文章、评论]
        - article   [一般来说会包含一个标题]
        - aside     [表示与页面内其余内容无关的内容，如侧边栏、标注框、广告]     
        - footer    [标识一个章节内容或者根节点的页脚，包含版权数据、友情链接等]
        - hgroup    [代表文章章节所属的多级别的目录，包含多个h1-h6元素]

- 本地存储
    - localStorage
    - sessionStorage
